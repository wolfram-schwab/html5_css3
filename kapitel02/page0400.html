Counter

Definition

Im Prinzip sind counter Variable, die von CSS zur Verfügung gestellt werden.

Initialisierung

Die Initialisierung erfolgt über die CSS-Eigenschaft

counter-reset : name

Inkrementierung

Die Inkrementierung erfolgt über die CSS-Eigenschaft

counter-increment : name

Immer dann, wenn die Auswertung des Stylesheets über diese Anweisung läuft, wird der Zähler um eins erhöht.

Darstellung

Die Darstellung erfolgt in einem Pseudo-Element über den Eigenschaftswert!

counter(name)

Beispiel

Der folgende Abschnitt – ausgezeichnet als <div> –  enthält die folgenden CSS-Regel:

counter-reset: counter01;

Die eingeschachtelten Absätze erhalten das Pseudo-Element ::before und die beiden Regeln:

counter-increment: counter01;
content : counter(counter01) ". ";

Anpassungen

Sowohl Startwert als auch die Schrittweite lassen sich individualisieren

Startwert

Der Startwert kann an den Namen des Zählers angehängt werden

counter-reset : name startwert

Schrittweite

Ähnlich erfolgt die Angabe der Schrittweite beim Inkrementieren

counter-increment : name schrittweite

Beispiel

Der folgende Abschnitt – ausgezeichnet als <div> –  enthält die folgenden CSS-Regel:

counter-reset: counter02 1000;

Die eingeschachtelten Absätze erhalten das Pseudo-Element ::after und die beiden Regeln:

counter-increment: counter02 10;
content :" Kd-Nr: " counter(counter02);

Gleichartige Elemente

Wenn mehrere geschachtelte, gleichartige Elemente einen Zähler mit demselben Namen initialisieren, erhält dennoch jedes Element seinen eigenen Zähler.

Mit der Eigenschaft counters(name, seperator) lassen alle Zähler in der Hierarchie aufrufen. Auf diese Weise lassen sich hierarchische Gliederungen realisieren.

Der Aufruf erfolgt jeweils von außen nach innen:
