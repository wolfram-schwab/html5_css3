Pseudo-Elemente

Mit Pseudoelementen lassen sich einem Dokument zum Einen zusätzliche Inhalte hinzufügen oder zum Anderen Textteile auszeichnen, die sonst nur schwer zugänglich sind

Zusätzlicher Inhalt

lässt sich sowohl am Anfang als auch am Ende eines Elements hinzufügen.
Die Syntax lautet:

selektor::before bzw. selektor::after

Hinzufügen lassen sich:

  • weiterer Text
  • Grafiken
  • Attributwerte

Der zusätzliche Inhalt wird über die Eigenschaft content definiert.

Text

wurde hinzugefügt.

Mit der CSS-Anweisung:

h1+p::before { content : "Absatz 1: "; font-weight : bold; }

Textuelle Inhalte sind immer gequotet "text" oder 'text'!

Sonderzeichen

Sonderzeichen werden mit einem \Backslash gefolgt vom utf-8 Code im laufenden Text kodiert

  • li::before { content : '\2714' }

s.a. Wikipedia verschiedene Symbole

Bilder

Bilder werden über den URL referenziert Euer  

Mit der CSS-Anweisung:

p>span::after { content : url(adresse); }

Attributwerte

Attributwerte werden über die Funktion attr(name) referenziert

PseudoElemente findet man bei MDN

Mit der CSS-Anweisung:

p>a::after { content : "("attr(href)")"; text-decoration : none; color : black; }

Zusätzliche Elemente

Sowohl der erste Buchstabe als auch die erste Zeile eines Textes lassen sich auswählen;
Diese werden dann in ein zusätzliches Element eingeschlossen.

Während sich das erste Zeichen über ein <span>-Element leicht auszeichnen lässt, ist dies bei der ersten Zeile - zumindest mit CSS - nicht möglich.

Erstes Zeichen

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce ultricies neque non orci varius tempor. Aliquam vitae tortor hendrerit, venenatis quam quis, mollis enim. Sed vel volutpat turpis. Suspendisse neque tellus, semper ac posuere a, pellentesque at metus. Sed ex orci, condimentum sed sollicitudin non, maximus eget turpis. Aliquam non ante lectus. Praesent interdum dolor sed sapien ornare, ut venenatis nibh scelerisque.

Mit der CSS-Anweisung:

h1+p::first-letter { font-size : 3.0em; font-weight : bold; float : left; ...}
