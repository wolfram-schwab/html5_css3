Flex kein Flax

Wachsen, Schrumpfen

  1. section {
  flex-basis : 100px;
} 
  2. section {
  flex-basis : 500px;
} 
  3. section {
  flex-grow : 1;
} 
  4. #s05 {
    flex-grow : 10;
  }
flex-basis
legt die Basisbreite jedes Elements fest;
dies ist als Minimalbreite zu verstehen
flex-grow
legt den Wichtungsfaktor (positive, ganze Zahl) für das Wachstum fest;
die Breite des Containers wird ausgefüllt.
flex-shrink
legt den Wichtungsfaktor (positive, ganze Zahl) für das Schrunmpfen fest;
falls der Platz nicht reicht.

Alle zusammen: flex : 1 1 auto (grow, shrink, basis)
